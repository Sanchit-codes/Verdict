================================================================================
TECHNICAL FINDINGS
================================================================================

Architecture:
- Client: Next.js 15.5.14 (TypeScript, Tailwind CSS)
- Server: Flask with Gunicorn (Python 3.10)
- Validation: Multi-tier cascade (heuristics → embedding → HHEM)
- Models: Auto-download from HuggingFace on first use
- Models cached in-process for reuse

Validation Pipeline (executed during request):
1. Content Classifier - Determines if content is factual (instant)
2. Prompt Structure - Analyzes prompt intent and PII (1ms)
3. Prompt Injection - Detects injection patterns (1ms)
4. Heuristics - Context coverage, entity overlap (instant)
5. Embedding - Cosine similarity of embeddings (100-200ms on cold, instant on warm)
6. HHEM - Vectara hallucination evaluation (40-100ms on cold, instant on warm)

Decision Logic:
- Risk score aggregated from all validators with configured weights
- Threshold comparison determines: allow / block / regenerate / abstain
- Evidence includes breakdown of each validator contribution
