SECURITY LAYERS SUMMARY:

Layer 1 (Thinking):   ArmorIQ checks LLM's reasoning       <20ms ⚡ EARLY!
Layer 2 (Content):    HallucinationGuard validates text    50-100ms
Layer 3 (Execution):  ArmorIQ enforces action alignment    <10ms

═══════════════════════════════════════════════════════════════════════════════

PERFORMANCE:

Normal Flow (good intent):
  Ground Truth (5ms) → Gemini (1000ms) → Extract+ArmorIQ (15ms) 
  → Heuristics (5ms) → Embeddings (30ms) → HHEM (80ms) = ~1130ms total

Fast Blocking (bad intent):
  Ground Truth (5ms) → Gemini (1000ms) → Extract+ArmorIQ (15ms)
  → BLOCKED = ~1020ms (110ms SAVED by early check!)

═══════════════════════════════════════════════════════════════════════════════

GROUND TRUTH PERSISTENCE:

Session Start:
  guard._ground_truth_store = {}

User 1 asks: "Book flight to Paris"
  guard._ground_truth_store[session_1] = GroundTruthContext(task="book flight to Paris", ...)

User 2 asks: "Find hotels in Paris" (same session)
  task = guard._get_session_task(session_1)  # Returns "book flight to Paris"
  ArmorIQ compares new actions against THIS persistent ground truth

DANGEROUS PATTERNS CAUGHT BY ARMORIQ:

Task: "book flight"
Thinking: "DELETE FROM users"         → ❌ BLOCKED (SQL dangerous)
Thinking: "rm -rf /home"             → ❌ BLOCKED (filesystem dangerous)
Thinking: "exec(...)"                → ❌ BLOCKED (code execution)
Thinking: "DROP TABLE"               → ❌ BLOCKED (SQL dangerous)

Task: "book flight"
Thinking: "search flights database"  → ✅ ALLOWED (aligned)
Thinking: "query fare options"       → ✅ ALLOWED (aligned)
